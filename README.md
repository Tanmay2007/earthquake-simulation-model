# SeismoSim — Arduino-Based Earthquake Simulation Model

> An interactive web-based platform demonstrating how base isolation protects structures from seismic activity, built around a physical Arduino-powered earthquake simulator.

**Live Site:** [tanmay2007.github.io/earthquake-simulation-model](https://tanmay2007.github.io/earthquake-simulation-model/)

---

## Overview

SeismoSim is a physics project that physically simulates earthquake ground motion using an Arduino-controlled DC motor with an eccentric mass. Two building models are placed on the vibrating platform — one normal structure and one base-isolated structure — to demonstrate the effectiveness of seismic isolation in real time.

The accompanying website provides an interactive 3D model viewer, detailed explanation of the working principle, component breakdown, and a side-by-side observation comparison.

---

## The Physical Model

A DC motor with an offset nut/bolt weight is mounted on a wooden platform. When powered, the rotating eccentric mass creates mechanical imbalance, generating oscillating vibrations across the entire platform — simulating seismic ground motion.

Two identical 3-storey building frames (made from cardboard and ice cream sticks) are placed on the platform:

| Structure | Isolation | Behaviour |
|---|---|---|
| Normal | None — sits directly on platform | Full vibration transfer, visible sway at all floors |
| Base-Isolated | Rubber/sponge pads at 4 corners | Energy absorbed by pads, minimal movement |

---

## Physics Concepts

- **Vibrations** — Mechanical oscillations generated by the rotating eccentric mass
- **Damping** — Energy dissipation through rubber/sponge isolation pads
- **Resonance** — Risk of amplified oscillation in the normal structure at motor frequency
- **Energy Transfer** — Seismic energy flow from ground through structure

## Components

| Component | Role |
|---|---|
| Arduino Uno | Microcontroller — runs control logic, sends PWM signals |
| L298N Motor Driver | Receives PWM from Arduino, drives DC motor at variable speed |
| DC Motor | Generates vibrations via rotating eccentric nut/bolt weight |
| 16x2 LCD Display | Shows real-time motor status via I2C module |
| Rubber / Sponge Pads | Base isolators — absorb and dampen seismic energy |
| Cardboard + Ice Cream Sticks | Structural material for both building models |
| Jumper Wires | Electrical connections between all components |
| 9V Battery + Connector | Powers the Arduino and motor driver |
| Breadboard | Prototyping connections |

## Conclusion

Base isolation is an effective method to reduce earthquake damage and improve structural safety by decoupling the building from ground motion. This model demonstrates that principle physically and visually, making it accessible for educational demonstration.
